# ZhiXiongScrollForWPF
仿Mac的滚动条，滚动时显示，不滚动时隐藏，带动画效果

##  前言
在网上找了很久没有找到能实现这种效果的代码，能找到的代码都是各种bug，于是自己写了一个。

* 支持所有控件
* 需要注意的就是嵌套的Scroll需要实现PreviewMouseWheel事件，在该事件中向上传递触发事件
